# Personal Portfolio Website

## Project Overview
This project is a personal portfolio website built using HTML, CSS, and JavaScript.  
The purpose of this website is to showcase my background, skills, experience, education, and personal achievements.  
It also includes a photo gallery and spaces for future content such as certificates and project images.

## Features
- Personal introduction and biography  
- Soft skills and hard skills section  
- Project showcase with placeholder images  
- Work experience list  
- Certificate placeholders (4 slots)  
- Education timeline  
- Photo wall arranged in a free layout  
- Contact information (Email, LinkedIn, GitHub placeholder)  
- Responsive structure  

## Requirements
- Any modern web browser (Chrome, Edge, Firefox, Safari)  
- No frameworks required  
- Runs using only:  
  - **HTML5**  
  - **CSS3**  
  - **JavaScript**
